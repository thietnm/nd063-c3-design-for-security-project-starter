# Describe GuardDuty findings that were detected related to the brute force attack
We have 2 findings which are marked as EC2/SSHBruteForce. These brute force attacks are used to gain unauthorized access to your instance by guessing the SSH password.
# Research the AWS Guard Duty documentation page and explain how GuardDuty may have detected this attack - i.e. what was its source of information.
Guard Duty continuously analyzes network traffic and identifies patterns that deviate from normal behavior. Also, If it observes numerous failed SSH login attempts from a single IP address or multiple IP addresses within a short timeframe, it can flag this as suspicious.
